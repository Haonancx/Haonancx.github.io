# FSUX-BlogTheme

## Design, Code & Things in Between

FSUX-BlogTheme is a theme for me. Here is [my blog](http://fsux.me) which use this theme.

![About-me](https://i.imgur.com/4hHbGdR.jpg)

- 添加了对文章标签、年度的分类
- 中文阅读体验优化、文章内容引用版权声明
- 加入百度统计对访问量实时监控
- 引入缓存机制，再次链接速度难以想象
- 兼容 IE19+ 及所有主流浏览器
